# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-07-21

### Added

- Interactive Android build, install, launch, APK discovery, and packaging flows.
- Project configuration initialization and validation.
- Android application module and build variant detection.
- Non-interactive `build`, `run`, `apk`, `package`, `devices`, and `avds` commands.
- Global `adf` console command packaging.
- `adf wrapper` for generating pinned `adfw` and `adfw.bat` project launchers.
- `adf --version` and `adf version` commands.
